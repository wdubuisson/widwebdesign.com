Website Evaluation and Improvement Proposal: SEEK - May 2018

I assisted in a project for the Safe Environment for Every Kid organization, where we evaluated the current website, and developed a prototype for the new website. The project involved conducting research related to the current website, testing on the current site and our prototypes, making recommendations for a redesigned website, as well as recommendations for enhancements that can be made to the current site. Our full proposal can be found here.

For this project we:

  • Created a content inventory
  • Conducted a competitive analysis
  • Ran user testing and treejack testing
  • Conducted a search analysis
  • Developed personas, scenarios and user journeys
  • Recommended a new IA
  • Used Balsamiq Wireframes to create an initial prototype
  • Conducted usability testing on recommended IA and initial prototype
  • Created a refined prototype, conducted more testing, made more improvements
  • Developed and communicated strategies for accessibility, and both domestic and global business
  • Presented our findings and provided data driven support for our final product
