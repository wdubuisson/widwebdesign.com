Converted PSD file to Responsive Website - June 2019

One of my projects through the Skillcrush Front End Developer Blueprint, was to convert this PSD file into code, and make it responsive using Flexbox.

After setting up my workspace, I started by coding up the main sections. Starting with the large areas: Header, Main, Footer; and breaking it down further from there.

I went on to add the other sections and all my headings and paragraphs. And at this point, the structure of my homepage was complete.

After I finished the html for all of my pages, it was time to style them. I added flexbox properties to my CSS, and editing the width, margins, and padding on my pages. I was able to determine the measurements by using Ruler tool in photoshop.

Next, I was ready to make my site responsive. I began this process by adding breakpoints.
